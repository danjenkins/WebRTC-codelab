WebRTC Code Lab    

Browser

You'll need a WebRTC enabled browser, there are currently two browsers that support the WebRTC API

Google Chrome Stable

Mozilla Firefox Stable

The preference for this code lab is Google Chrome as we will be using Chrome DevTools which is a set web authoring and debugging tools built into Google Chrome. The DevTools provide you access into the internals of the browser and the associate web application.
