Code Editor

You are a developer so you've more than likely already got your favourite code editor.

  • VIM You are a minimalist at heart, but you value raw power
  • Emacs You wanted the most powerful, and most extensible editor on Earth, and you can write a formal mathematical proof that Emacs is that editor
  • Nano All you need is a working text editor and Nano is just that. No frills, no bells, no whistles. Just rock solid functionality.
  • Nodepad++ You like the simplicity of Notepad, but you wanted a bit more.

This code lab has mainly HTML5 and Javascript code editing.
